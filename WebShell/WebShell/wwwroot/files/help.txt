---------- Список команд ----------

webhelp
Список команд WebShell
-----------------------------------

dbconf
Установка настроек соединения с базой данных
--- Параметры ---
 -s ServerName      ---> Выбор сервера
 -d DataBaseName ---> Выбор базы данных"
 -e                           ---> Использование доверительного соединения вместо логина и пароля(Установлен по умолчанию)
 -u UserName         ---> Логин пользователя для входа в SQL Server 
 -p Password           ---> Пароль пользователя для входа в SQL Server
-----------------------------------

sql Query
Отправка SQL запроса на сервер
Требуются установленные параметры dbconf -s и dbconf -d
